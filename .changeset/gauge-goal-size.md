`gauge.goal` gains a `size` (`small` / `medium` / `large`, default `medium`) controlling its footprint, so several goals can sit side by side in a row without crowding. `small` swaps the status badge for a compact status dot; `large` emphasizes a single goal. The ring, value, and stroke scale together.
